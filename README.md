# 🌙 AI Night Blindness Assistance System

## 📌 Overview
The **AI Night Blindness Assistance System** is a real-time assistive solution designed to help individuals with **low vision or night blindness** navigate safely using **Artificial Intelligence and Computer Vision**.

This system acts as a **smart visual assistant**, converting real-world visual information into **audio feedback and alerts**, enabling users to better understand their surroundings—especially in low-light conditions.

Built using **Python (Flask)** and a responsive **web interface**, the application processes live camera input and delivers real-time insights.

---

## 🚀 Key Features

### 🔍 Real-Time Object Detection
- Detects objects such as people, chairs, tables, and obstacles  
- Uses deep learning models like YOLO  
- Displays bounding boxes on live video feed  

---

### 📏 Distance Estimation
- Estimates how far objects are from the user  
- Helps in avoiding nearby obstacles  

---

### 🧭 Direction Detection
- Identifies object position:
  - Left  
  - Center  
  - Right  
- Assists users in making navigation decisions  

---

### 🔊 Voice Alert System
- Converts detections into real-time audio feedback  
- Example:
  - “Person detected in front of you”  
  - “Chair on your left”  

---

### ⚠️ Smart Alert Dashboard
- Displays:
  - Object name  
  - Distance  
  - Direction  
  - Warning level  
- Provides both **visual and auditory assistance**

---

### 🚨 Obstacle Warning System
- Detects critical obstacles like:
  - Walls  
  - Furniture  
- Triggers alerts when objects are too close  

---

## 🧠 Advanced Features

### 👥 Crowd Detection
- Detects multiple people in the environment  
- Alerts user in crowded areas  
- Example: “Crowded area ahead”  

---

### 🪧 Text Reading (OCR)
- Reads text from surroundings  
- Useful for:
  - Exit signs  
  - Labels  
  - Room numbers  
- Example: “Exit sign detected”  

---

### 💰 Currency Detection
- Identifies Indian currency notes and coins  
- Helps users handle money independently  
- Example: “₹500 note detected”  

---

### 🚦 Traffic Signal Detection
- Detects traffic lights and crossings  
- Provides safety alerts  
- Example: “Red signal detected. Please stop.”  

---

## 🛡️ Safety & Assistance Features

### 🚨 SOS Emergency System
- Sends emergency alerts with location  
- Notifies registered contacts via SMS  

---

### 🎙 Voice Command Control
- Hands-free interaction using voice commands:
  - Start/Stop detection  
  - Activate SOS  
  - Describe surroundings  

---

### 🌙 Low-Light Enhancement
- Enhances visibility in dark environments  
- Improves detection accuracy at night  

---

## 🛠️ Technology Stack

- **Backend:** Python, Flask  
- **AI/ML:** YOLO, OpenCV  
- **OCR:** EasyOCR / Tesseract  
- **Frontend:** HTML, CSS, JavaScript  
- **Voice:** Text-to-Speech, Speech Recognition  
- **APIs:** Twilio (SMS), GPS APIs  

---

## ⚙️ System Workflow

1. Capture live video from camera  
2. Process frames using AI models  
3. Detect objects and estimate distance  
4. Identify object direction  
5. Generate alerts and voice feedback  
6. Display results in real-time dashboard
